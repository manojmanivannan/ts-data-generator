# Dimension Generators

Dimensions are categorical or continuous columns generated by infinite iterator functions.

## Helper Functions

Located in `ts_data_generator.utils.functions`.

### `random_choice(values)`
Randomly picks an element from a list.

### `random_int(start, end)`
Randomly picks an integer between `start` and `end`.

### `random_float(start, end)`
Randomly picks a float between `start` and `end`.

### `constant(value)`
Returns the same value indefinitely.

### `ordered_choice(values)`
Cycles through a list of values in order.

### `auto_generate_name(prefix)`
Generates unique names with a given prefix (e.g., `cat_1`, `cat_2`).

## Custom Dimensions

You can provide any generator function:

```python
def my_custom_gen():
    while True:
        yield "A" if random.random() > 0.5 else "B"

dg.add_dimension("my_dim", my_custom_gen())
```
